# AI Book Companion Project

## Overview
This project is an AI Book Companion built using **Next.js**, **Vapi**, and **ElevenLabs**. It aims to provide users with an interactive and engaging reading experience.

## Features
- **Interactive Book Recommendations:** AI-driven suggestions based on reading history.
- **Voice Interaction:** Using ElevenLabs for a voice interface that assists users.
- **Dynamic Content:** Real-time updates and information about books.

## Setup Instructions
### Prerequisites
- Node.js >= 14.x
- npm >= 5.x
- Access to ElevenLabs API

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/AKDev32/Bookified.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Bookified
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Configure environment variables for API keys and other necessary settings.
5. Start the development server:
   ```bash
   npm run dev
   ```

### Usage
- Open your browser and navigate to `http://localhost:3000` to view the project.
